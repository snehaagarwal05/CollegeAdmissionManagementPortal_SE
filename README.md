College Admission Management Portal

########--------- Project Overview ----------########

The College Admission Management Portal is a web-based application designed to digitalize and streamline the college admission process. It provides separate dashboards for students and administrators, enabling efficient application submission, tracking, verification, and admission management.
This system eliminates manual paperwork, reduces errors, and improves transparency between applicants and the institution.

########--------- Objectives ----------########

    Automate the college admission process
    Provide a secure login system for students and admin
    Allow students to submit and track applications online
    Enable administrators to verify applications and manage admissions
    Improve efficiency, accuracy, and accessibility

########---------- User Roles ----------########

    1. Student
    
        Register and log in
        Fill and submit admission application
        Upload required documents
        View application status
        Proceed to application fee payment
    2. Admin
    
        Secure admin login
        View all student applications
        Verify and approve/reject applications
        Manage admission status

#########-------- How to Run the Project ---------########

1. Clone the Repository
    git clone https://github.com/your-username/CollegeAdmissionManagementPortal_SE.git
2. Backend Setup
    cd backend
    npm install
    npm install pdfkit
    npm install razorpay
    npm start

4. Frontend Setup
    cd frontend
    npm install
    npm run dev
